## 💫 About Me

ECE undergrad passionate about hardware design and intelligent systems — I work across RTL design, FPGA implementation, edge ML, and IoT. I've built projects ranging from a **4-bit ALU synthesized on an Artix-7 FPGA** to an **ML-based signal denoising system deployed on Raspberry Pi**.

I enjoy understanding how things work at the silicon level and building systems that bridge hardware and intelligence.

- 🔭 Currently working on **FPGA-based hardware security systems & edge ML deployment**
- 👯 Looking to collaborate on **VLSI design, RTL projects, and embedded ML applications**
- 🌱 Learning **CUDA programming** on RTX 5050 (Blackwell) & preparing for **GATE 2027**
- 💬 Ask me about **Verilog, FPGA design, signal processing, IoT, Raspberry Pi**
- ⚡ Fun fact: **I build things that run on silicon — from RTL code to edge AI on a $35 computer**
